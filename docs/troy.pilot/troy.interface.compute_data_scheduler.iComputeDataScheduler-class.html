ComputeDataScheduler (CDS)

The CDS is a troy-internal object which provides scheduling capabilities to Troy. In particular, it will schedule compute_data units over a set of compute_data pilots. To do that, the scheduler implementation (adaptor) will need to pull various information from the backend.
