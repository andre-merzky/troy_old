ComputePilot (PilotJob)

This is the object that is returned by the ComputePilotService when a new ComputePilot is created based on a ComputePilotDescription.

The ComputePilot object can be used by the application to keep track of ComputePilots that are active.

A ComputePilot has state, can be queried, can be cancelled and be re-initialized.
