DataUnitService (DUS)

The DataUnitService is the application's interface to submit DataUnits to the Pilot-Manager in the P* Model.

It can provide the application with a list of DataUnits that are managed by the Pilot-Manager.

The DataUnitService is linked to a DataPilotService for the actual execution of the DataUnits.
