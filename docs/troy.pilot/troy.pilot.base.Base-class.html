Class Base

source code

All Troy API classes inherit from this base class, which provides interfaces to adaptor management and to the management of class instance data.

In particular, this base class provides the following class members to all API classes:

  • engine_ : a reference to the engine singleton, which manages adaptors and call forwarding
  • adaptors_ : a sorted list of used adaptors, which *can* be used by the engine to optimize adaptor invocation (previously successful adaptors are most likely to succeed again)
  • idata_ : a dict of dicts, for class instance state data management. The dict under idata_['api'] holds the actual class instance data (documented for each class), all other keys hold adaptor level instance data, for example to cache connections. The keys for those adaptor instance data SHOULD be the adaptor name. idata_ SHOULD not be accessed directly, but via get_idata() and set_idata(),

The accessor methods are marked as private, as they are not part of the API. They are intended to be used by the engine and the adaptors.
