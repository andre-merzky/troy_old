ComputeDataUnit.

The ComputeDataUnit is a handle to a unit of (compute and/or data) workload. It is the object that is returned by the ComputeDataUnitService when a new ComputeDataUnit is created based on a ComputeDataUnitDescription.

The ComputeDataUnit object can be used by the application to keep track of ComputeDataUnits that are active.

A ComputeDataUnit has state, can be queried and can be cancelled.
