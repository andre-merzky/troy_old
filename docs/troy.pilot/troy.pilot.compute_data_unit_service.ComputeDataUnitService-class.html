ComputeDataUnitService.(CDUS)

The ComputeDataUnitService provides the functionality of both the ComputeUnitService and the DataUnitService -- as such, it can enact a combination of ComputeUnits and DataUnits, i.e. the ComputeDataUnits, on a set of ComputePilots and DataPilots.

The implementation of the ComputeDataUnitService is generally within Troy -- as such, it represents and facilitates the application level scheduling capabilities of Troy. The CDUS will use Troy level or application provided scheduling algorithms to control the distribution of the work nits over the set of participating pilot dervices and their pilots.

Properties:

 - id:
   The returned ID can be used to connect to the CDUS instance later
   on, for example from within a different application instance.
   type: string (url)

list_compute_data_units(self)

source code 

list managed ComputeUnits.

Return value: A list of ComputeDataUnit IDs

The returned list can include units which have not been created by this CDUS instance. The list may be incomplete, and may not include units created by the CDUS. There is no guarantee that units in the returned list can in fact be reconnected to. Also, an inclusion in the list does not have any indication about the respective unit's state.
