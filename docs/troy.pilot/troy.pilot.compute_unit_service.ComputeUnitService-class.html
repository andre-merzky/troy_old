ComputeUnitService (CUS)

The ComputeUnitService is the application's interface to submit ComputeUnits to the Pilot-Manager in the P* Model. It can provide the application with a list of ComputeUnits that are managed by the Pilot-Manager.

The ComputeUnitService is linked to a set of ComputePilots for the actual execution of the ComputeUnits.
