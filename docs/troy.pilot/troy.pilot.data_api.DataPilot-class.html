DataPilot (PilotStore)

This is the object that is returned by the DataPilotService when a new DataPilot is created based on a DataPilotDescription.

The DataPilot object can be used by the application to keep track of DataPilots that are active.

A DataPilot has state, can be queried, can be cancelled and re-initialized.
