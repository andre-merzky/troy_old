11      A DataPilotDescription (DPD) describes a L{DataPilot} to be 
12      submitted to a resource.  A well defined set of attributes can be set on 
13      a DPD to specify the pilot's properties, and the resource requirements 
14      for the pilot: 
15   
16        - 'size': 
17            - minimum size of storage the pilot is expected to manage at any 
18              point in time (usually translates to number of bytes of storage 
19              allocated by the pilot). 
20            - type   : integer 
21            - default: 1024 * 1024 * 1024 (1GB) 
22   
23        - 'project': 
24            - the project ID used for accounting of the pilot's resource 
25              consumption. 
26            - type   : string 
27            - default: "" 
28   
29        - 'candidate_hosts': 
30            - a list of hostnames on any of which the pilot may be operate. 
31            - type   : string 
32            - default: "" 
33   
34        - 'wall_time_limit': 
35            - the maximum time the pilot is expected to operate, in hours. 
36              If unspecified, the pilot is assumed to live forever. 
37            - type   : float (number of hours) 
38            - default: "" 
39   
40        - 'start_time': 
41            - the point in time when the pilot is expected to become active. 
42            - type   : time (number of seconds since epoch) or class time.struct_time 
43            - default: "" 
44   
45        - 'contact': 
46            - a (email, sms, IM) contact URL to notify on pilot state changes. 
47            - type   : string 
48            - default: "" 
49   
50   
51   
52      Affinity: 
53   
54      These labels should get assigned by the backend, but are exposed on API 
55      level for the benefit of application level schedulers:: 
56   
57        - 'affinity_datacenter_label': 
58            - pilots sharing the same label are located in the same data 
59              center          . 
60            - type   : string 
61            - default: "" 
62   
63        - 'affinity_machine_label': 
64            - pilots sharing the same label are located on the same machine. 
65            - type   : string 
66            - default: "" 
67   
68   
69      Note that the DPD does not describe how the pilot is instantiated in 
70      detail (e.g.executable name of pilot instance) -- that is left to the 
71      backend to decide, and may (or may not) be configurable out-of-band.
