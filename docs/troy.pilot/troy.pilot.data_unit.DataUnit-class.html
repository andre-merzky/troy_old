Class DataUnit

source code

DataUnit.

This is the object that is returned by the DataUnitService when a new DataUnit is created based on a DataUnitDescription.

The DataUnit object can be used by the application to keep track of DataUnits that are active.

A DataUnit has state, can be queried and can be cancelled.
