DataUnitDescription (DUD)

The DataUnitDescription describes a set of data items which are expected to share locality properties. The data items are referenced by URLs, so, in some sense, the DUD is a list of URLs.

The description is used via a DataUnitService to instantiate DataUnits, i.e. physical data representations, which are managed by a DataPilot.

Currently, no directory URLs are supported.
